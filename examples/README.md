# GBLIN-MCP Examples

## demo-agent.ts

Complete end-to-end demo of all 6 GBLIN MCP tools against Base mainnet.
Zero transactions broadcast — read-only smoke test.

```bash
npm run demo
```

## elizaos-character.json

ElizaOS character configuration with plugin-gblin pre-loaded.
Copy to your ElizaOS project and install plugin-gblin:

```bash
bun add plugin-gblin
```
